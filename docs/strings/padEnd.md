<h3 id="padendstring-length-substr"><code>padEnd(string, length, [substr=' '])</code></h3>

PadEnd pads the end of a string.

#### Arguments
1. `string` *(string)*:
2. `length` *(number)*:
3. `[substr=' ']` *(string)*:

#### Returns
*(string)*:

#### Example
```js
// if no `substr` is provided, it pads the string w/ spaces
const result = strings.padEnd('abcabc', 9);
console.log(result);
> abcabc
```
