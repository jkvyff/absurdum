<h3 id="maparray-func"><code>map(array, func)</code></h3>

Map iterates over an array of values and applies a function to each value

#### Arguments
1. `array` *(Array)*:
2. `func` *(Function)*:

#### Returns
*(Array)*: The array of mutated values

#### Example
```js
const result = arrays.map([1, 2, 3, 4], (x) => x + 2);
console.log(result)
> [ 3, 4, 5, 6 ]
```
