<h3 id="reversearray"><code>reverse(array)</code></h3>

Reverses the order of the values of an array

#### Arguments
1. `array` *(Array)*: input array

#### Returns
*(Array)*: input array in reversed order

#### Example
```js
const result = arrays.reverse([1, 2, 3, 4]);
> [4, 3, 2, 1]
```
