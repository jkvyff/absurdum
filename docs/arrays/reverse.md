<h3 id="reversearray"><code>reverse(array)</code></h3>

Returns an array in reverse order

#### Arguments
1. `array` *(Array)*:

#### Returns
*(Array)*: The array in reversed order

#### Example
```js
const result = arrays.reverse([1, 2, 3, 4]);
> [4, 3, 2, 1]
```
