<h3 id="uniquearray"><code>unique(array)</code></h3>

Removes all duplicate items of an array

#### Arguments
1. `array` *(Array)*: input array

#### Returns
*(Array)*: an array of unique values

#### Example
```js
const result = arrays.unique([2, 1, 2]);
console.log(result);
> [2, 1]
```
