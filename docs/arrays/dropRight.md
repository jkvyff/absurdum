<h3 id="droprightarray-n1"><code>dropRight(array, [n=1])</code></h3>

Returns an array with n items dropped from the end.

#### Arguments
1. `array` *(Array)*:
2. `[n=1]` *(number)*:

#### Returns
*(Array)*: The input array sans the dropped items

#### Example
```js
const result = arrays.drop([1, 2, 3], 2);
console.log(result);
> [1]
```
